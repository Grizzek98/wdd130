BeSquare

BeSquare is a simple 2D platformer game I worked on as project for a class over the course of a two-week sprint. It was built in the Unity Game Engine, and features a main menu, level select menu, and three short levels.

Game mechanics include smooth movement and jumping, enemies with basic tracking, environmental hazards (pits and spikes). I had a lot of fun working on this game by myself, and learned a lot in the process. I got to figure out how to make some pretty, albeit simple, graphics. I even got to learn a little bit of Unity's particle engine, which upped the aesthetic of the game immensely.

Future plans include sounds, music, additional levels, additional enemies/hazards, and perhaps a player dash and other powerup-type abilities.

Applicable technologies: C# language, Unity Game Engine, Visual Studio Code.

Check out the repo on GitHub for more info, including a video demo of the game.
